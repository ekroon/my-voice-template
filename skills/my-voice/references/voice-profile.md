# Voice Profile

> This file will be generated during the voice capture process.
> Start the my-voice agent and it will guide you through writing samples
> and produce your personalized voice profile here.
>
> Run: `/agent my-voice`

## How it works

1. The agent asks you to write 5-8 short pieces in different contexts
2. After each piece, it shares observations about your style
3. When enough samples are collected, it generates your full profile here
4. You review and approve the profile
5. From then on, the skill and agent use this profile to match your voice
